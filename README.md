# Campus Resume Builder

中文优先、桌面端优先的网页版校园简历编辑器。

## 项目简介

Campus Resume Builder 面向学生和校招求职者，目标是用更低的排版门槛完成一份结构清晰、观感正式的中文简历。

当前版本聚焦 MVP：左侧模块化填写，右侧实时预览，支持模块显隐、预览区拖拽排序，以及通过浏览器打印导出 PDF。

## 核心功能

- 中文优先的校园简历编辑体验
- 左侧编辑，右侧实时预览
- 预览区支持模块拖拽排序
- 模块显示开关与编辑区联动
- 本地 `localStorage` 持久化
- A4 预览与浏览器打印 / PDF 导出
- 超出一页时允许自然分页

当前支持的简历模块：

- 个人信息
- 教育经历
- 项目经历
- 实习经历
- 校园经历
- 技能特长
- 获奖 / 证书

## 技术栈

- React 18
- TypeScript
- Vite
- Zustand
- dnd-kit
- Vitest + React Testing Library
